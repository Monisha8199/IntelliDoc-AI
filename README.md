# 🤖 IntelliDoc-AI

An AI-powered document analysis platform built using **Python**, **Streamlit**, **Groq AI**, **FAISS**, and **LangChain**.

## 🚀 Features

- 📄 AI Document Summarization
- 💬 AI Chat with Documents
- 🌐 Document Translation
- 📊 Document Analytics
- 🔑 Keyword Extraction
- 😊 Sentiment Analysis
- 📝 AI Quiz Generator
- 🎴 Flashcard Generator
- 📄 Resume Analyzer
- 📑 Document Comparison
- 🗂️ History Management

## 🛠️ Tech Stack

- Python
- Streamlit
- Groq API
- LangChain
- FAISS
- SQLite
- PyPDF2
- python-docx

## ▶️ Installation

```bash
git clone https://github.com/Monisha8199/IntelliDoc-AI.git
cd IntelliDoc-AI
pip install -r requirements.txt
streamlit run app.py
```
